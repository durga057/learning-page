# Learning Page

A premium academic resource portal designed for engineering students. This platform provides a centralized hub for syllabus information, lecture notes, video tutorials, and interactive learning trackers.

## 🚀 Features

- **Modern UI/UX**: A sleek, dark-themed portal with glassmorphism effects and smooth animations.
- **Syllabus Tracker**: Dynamic branch-wise syllabus viewer with PDF preview and download options.
- **Dark Mode**: Support for both light and dark themes with persistent user preferences.
- **Profile Management**: Local student profile dashboard with editable details and data persistence.
- **Resources**: Easy access to PDFs, including notes, syllabus, and previous question papers.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technologies Used

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6)
- **Styling**: Google Fonts (Outfit), Font Awesome 6 Icons
- **Data Management**: JSON for syllabus database
- **Deployment**: GitHub Pages
